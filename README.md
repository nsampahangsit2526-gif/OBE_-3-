คู่มือเครื่องมือสร้าง มคอ.3 รายวิชา (tqf3-generator.html)

ระบบสร้าง รายละเอียดของรายวิชา (มคอ.3) สำหรับหลักสูตรศิลปศาสตรบัณฑิต สาขาวิชาสื่อสารการกีฬา (ปรับปรุง พ.ศ. 2568) คณะศิลปศาสตร์ มหาวิทยาลัยการกีฬาแห่งชาติ วิทยาเขตกรุงเทพ

แนวคิด: เปิดมาเกือบเสร็จ อาจารย์ผู้สอนเหลือเติมนิดเดียว — ข้อมูลรายวิชา (คำอธิบาย, CLO, การเชื่อม PLO) ดึงจากเล่มหลักสูตรให้อัตโนมัติ อาจารย์กรอกแค่แผนการสอนรายสัปดาห์และน้ำหนักการประเมิน


สถานะปัจจุบัน

เวอร์ชัน ตัวอย่าง (prototype) — มีข้อมูล 3 รายวิชาคละประเภทไว้ให้ทดลอง:

รหัสชื่อวิชาประเภทค้ำ PLO หลักนศ 011064หลักและทฤษฎีการสื่อสารพื้นฐานวิชาชีพ · ทฤษฎีล้วนPLO1มศ 051089พัฒนาทักษะการฟัง–พูดภาษาอังกฤษฯเฉพาะด้านบังคับ · มีปฏิบัติPLO2นศ 062025การถ่ายภาพกีฬาเลือกด้านผลิตสื่อ · มีปฏิบัติPLO3

เมื่อเคาะแนวเรียบร้อยแล้ว จะลงข้อมูลครบทั้ง 73 วิชาเฉพาะ (หรือ 115 วิชาทั้งเล่ม) และต่อปุ่มดาวน์โหลดเป็นฟอร์มราชการเต็มรูป


วิธีใช้งาน


เปิดไฟล์ tqf3-generator.html ด้วยเบราว์เซอร์ (ดับเบิลคลิกได้เลย ไม่ต้องต่อเซิร์ฟเวอร์)
เลือกรายวิชา จาก dropdown ด้านบน
กรอก ภาคการศึกษา/ปี และ ชื่ออาจารย์ผู้สอน
ระบบเติมให้อัตโนมัติ: ข้อมูลทั่วไป (หมวด 1), คำอธิบายรายวิชา (หมวด 3), CLO + การเชื่อม PLO (หมวด 4)
อาจารย์เติมส่วนที่เป็นของเทอมจริง:

หมวด 5 แผน 15 สัปดาห์ — เติมหัวข้อแต่ละสัปดาห์ และระบุ CLO ที่ครอบคลุม (ชั่วโมง + สัปดาห์ปฐมนิเทศ/สอบ ใส่ตั้งต้นให้แล้ว)
หมวด 6 การประเมิน — ปรับน้ำหนัก % ให้รวมเป็น 100 (มีตารางตั้งต้นให้)



กด บันทึกร่าง (เก็บในเครื่อง) หรือ ดาวน์โหลด มคอ.3 (Word)



โครงสร้าง มคอ.3 ในเครื่องมือ

หมวดเนื้อหาที่มา1ข้อมูลทั่วไป (รหัส ชื่อ หน่วยกิต ชั้นปี หมวด/กลุ่มวิชา)ดึงอัตโนมัติ3ลักษณะและการดำเนินการ — คำอธิบายรายวิชา (ไทย/อังกฤษ) + ชั่วโมงรวมดึงอัตโนมัติ4ผลลัพธ์การเรียนรู้ (CLO) และการเชื่อมโยง PLOเติม PLO อัตโนมัติ · แก้ได้5แผนการสอน 15 สัปดาห์อาจารย์กรอก (มีค่าตั้งต้น)6การวัดและประเมินผล (น้ำหนัก %)อาจารย์กรอก (มีค่าตั้งต้น)


หมายเหตุ: ยังไม่รวมหมวด 2 (กลยุทธ์การสอน/การพัฒนา) และหมวด 7 (การปรับปรุง) — เพิ่มได้ถ้าต้องการ




การเติม PLO อัตโนมัติทำงานอย่างไร

ระบบเดา PLO ที่แต่ละ CLO ค้ำ จาก 2 แหล่ง แล้วกรองด้วยชั้นปี:

1) แท็กท้าย CLO (วิชาเฉพาะหลายตัวเขียนไว้ในเล่ม)


(Knowledge) → PLO1
(Skill/Skills) → PLO3 (หรือ PLO2 ถ้าเป็นวิชาภาษาอังกฤษ)
(Character) → PLO4
(Ethic) → PLO4


2) คำสำคัญในข้อความ (กรณีไม่มีแท็ก)


ภาษาอังกฤษ / ฟัง พูด อ่าน เขียน → PLO2
ผลิต / สร้างสรรค์ / ออกแบบ / ถ่ายภาพ / ตัดต่อ → PLO3
จริยธรรม / จรรยาบรรณ / รู้เท่าทัน / รับผิดชอบ → PLO4
ไม่เข้าเงื่อนไขใด → PLO1


3) กรองตามชั้นปี (YLO → PLO จากตาราง 9)


ปี 1 → PLO 1, 2, 4
ปี 2 → PLO 1, 2, 3, 4
ปี 3 → PLO 2, 3, 4
ปี 4 → PLO 2, 3, 4



แก้ด้วยมือได้ — แตะที่ชิป PLO ใต้แต่ละ CLO เพื่อเปิด/ปิดการเชื่อมโยง ค่าที่ระบบเติมเป็นเพียงค่าตั้งต้น




ความสัมพันธ์ มคอ.3 ↔ มคอ.5

เครื่องมือนี้ออกแบบให้ มคอ.3 และ มคอ.5 ใช้ โครง CLO→PLO ชุดเดียวกัน:

เลือกวิชา → มคอ.3 (วางแผน CLO/PLO/วิธีวัด) → สอนจบ → เปิดไฟล์เดิมเติมผล → มคอ.5

เพราะ มคอ.5 อ้างโครงของ มคอ.3 อาจารย์จึงไม่ต้องพิมพ์ CLO/PLO/วิธีวัดซ้ำ — กรอกแค่ผลลัพธ์ ตัวเลขจะม้วนเข้า PLO อัตโนมัติ เชื่อมต่อกับฐานข้อมูล Excel และ Dashboard ที่มีอยู่ได้
